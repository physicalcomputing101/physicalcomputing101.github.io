Welcome to Physical Computing 101

To get started working with Physical Computing requires knowledge about three distinct fields (electronics, programming and making). And in this world of the Internet, it is not difficult to find resources for each of these fields. However, how do you know if a resource is of good quality or a waste of time? On this site we try to touch on all the relevant aspects of Physical Computing.

The Modules

This site consists of a number of modules that each describe different parts of Physical Computing.

  • Physical computing: What is physical computing and why is it relevant today? What is required to start working with physical computing, examples of physical computing and the history of physical computing.

  • Sensing Signals: How does microcontrollers sense signals? To understand that requires understanding the basics of electricity, Ohm's and Kirchhoff's law. Furthermore an understanding of some of the most basic electronics components is required such as resistors, capacitors and diodes.

How can you help?

As this site is meant for all to use, students and teachers alike, we've chosen to host it on GitHub because it is easy to report issues with the site if some information is either not correct or explained properly. Any such issues can be reported here and if we agree with the issue we will fix it.
